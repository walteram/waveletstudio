Wavelet Studio é mais que apenas wavelets: é uma ferramenta para processamento de sinais em diversos campos. Conectando os blocos de processamento da biblioteca você pode criar, avaliar e testar modelos completos com poucos cliques.

Conheça alguns dos blocos disponíveis:

- Geradores de Sinais através de funções seno, cosseno, triangular, rampa, serra, quadrada, etc.
- FFT e IFFT
- DWT e IDWT
- Convolução
- Sub-amostragem e super-amostragem
- Operações escalares
- Operações entre sinais
- Importação e exportação de sinais em arquivos

E a cada versão estamos aumentando o número de blocos disponíveis. Se você é um desenvolvedor, ainda pode criar blocos próprios e mais adequados à sua necessidade. Senão basta solicitar a implementação de um bloco através deste link.

Consulte a documentação para informações detalhadas sobre cada um dos blocos.

Wavelet Studio is more than just wavelets: it is a robust tool for signal processing in general.
Using the available blocks you can create, evaluate and test complex models with a few clicks.

See below some of the available blocks:

- Signal generators through sine, cosine, triangle, ramp, saw, square and other functions
- FFT and IFFT
- DWT and IDWT
- Convolution
- Downsampling and Upsampling
- Scalar operations (sum, subtract, multiply, divide)
- Sample-based operations  (sum, subtract, multiply, divide)
- File import and export

We are increasing the number of available blocks. If you're a developer you can create your own blocks too. Otherwise just request the implementation of a block through this link.

For detailed information on each block, see the documentation.
